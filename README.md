## Getting Started

Follow these steps to get a local copy up and running.

### Prerequisites

- npm

```sh
npm install npm@latest -g
```

### Installation

2. Clone the repo

```sh
git clone https://github.com/maismin/hash-history-basketball-league.git
```

3. Install NPM packages

```sh
npm install
```

6. Start the web app

```sh
npm run start
```
